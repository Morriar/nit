ImSto: a little image storage
=============================

Requirements
------------

 * MongoDB (GridFS)
 * Python 2.7 + [pymongo][pymongo]
 * [ImageMagick][ImageMagick]
 * Nginx + [uWSGI][uWSGI]
 * jpegoptim (option)
